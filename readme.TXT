HOW TO LAUNCH THE PROJECT

1. Install Node JS

2. Insrall Mongodb Community Server
   -> Create a Database : StudentHub
   -> Create a Collection : Students

3. Open the project folder in VS Code

4. Open Ternminal in VS Code
   ->Click Terminal in menu bar -> click new terminal

5. Run Node commad to start the node server
   ->Command : node api.js

6. Open any webpage in the "FrondEnd folder"#
   ->Right Click -> Open with live sever

Node : To use Live Server, you have to install "Live Server" Extention in VS Code
url : https://www.youtube.com/watch?v=9kEOkw_LvGU
